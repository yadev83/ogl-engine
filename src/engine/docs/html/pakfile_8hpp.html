Detailed Description

Définit une interface pour lire le contenu des fichiers PAK. Le format PAK étant pensé spécifiquement pour le moteur de jeu, il est utilisé pour bundler les assets par la toolchain de compilation. On peut ensuite charger les .PAK, et récupérer les assets en mémoire.
