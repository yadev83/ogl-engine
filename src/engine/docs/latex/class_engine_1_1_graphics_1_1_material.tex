\doxysubsection{Detailed Description}
La classe \doxylink{class_engine_1_1_graphics_1_1_material}{Material} définit un "{}material"{} graphique utilisé pour le rendu de meshes texturés. 

Un \doxylink{class_engine_1_1_graphics_1_1_material}{Material} doit nécessairement avoir un \doxylink{class_engine_1_1_graphics_1_1_shader}{Shader} associé pour le rendu. Il peut avoir une texture aussi. En plus de cela, le \doxylink{class_engine_1_1_graphics_1_1_material}{Material} possède une "{}color"{} (qui peut être utilisée pour un rendu flat si pas de texture ou en mix avec la texture)
