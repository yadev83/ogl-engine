La méthode on\+Fixed\+Update est appelée dans le cycle de vie de l\textquotesingle{}app et gère les collisions, leur résolution, ainsi que la gestion de la dynamique (application de vitesses, mouvements).

Facteur d\textquotesingle{}echelle pour le système physique. 

La simulation physique est basée sur les calculs qu\textquotesingle{}on pourrait faire dans le monde réel. Mais un mètre dans la vraie vie n\textquotesingle{}est pas équivalent à un mètre dans l\textquotesingle{}application.

Ce facteur permet de dire combien d\textquotesingle{}unités (dans le jeu) sont équivalentes à un mètre (dans la vie) Par défaut, la valeur est de 100u/m, si 1 unité = 1 pixel, ça veut dire que 100 pixels = 1 mètre.
